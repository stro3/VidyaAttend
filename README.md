# Firebase Studio

This is a NextJS starter project created in Firebase Studio.

To get started, take a look at `src/app/page.tsx`.

## Deploying to Firebase

This project is configured for deployment with [Firebase App Hosting](https://firebase.google.com/docs/app-hosting).

1.  **Create a Firebase Project** in the Firebase Console.
2.  Navigate to the **App Hosting** section and create a new backend.
3.  Connect your Git repository (e.g., from GitHub) where this code is stored.
4.  Follow the prompts to configure your build and deployment settings.
5.  Once set up, App Hosting will automatically deploy your app whenever you push changes to your main branch.
